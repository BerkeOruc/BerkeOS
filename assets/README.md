# BerkeOS Assets

This directory contains project assets including screenshots.

## Screenshots

Screenshots should be placed in `screenshots/` directory.

### Required Screenshots

The following screenshots are needed for documentation:

- `boot.png` - System boot screen
- `shell.png` - Command line shell
- `neofetch.png` - System information display
- `editor.png` - Text editor interface
- `berkefs.png` - File system browser
- `calc.png` - Calculator application
- `sysinfo.png` - System information panel

### Screenshot Requirements

- Format: PNG
- Resolution: 1280x720 (minimum)
- Include window decorations where applicable
